# Bit Manipulation

## Bitwise Operators in C

### Definition
> Bit is the smallest unit (to store data) in computer memeory. A byte is 8
> bits.
> Data is stored in bitwise operation in computer memory.
### Bitwie Operators:
+ `&`	-	AND
+ `|`	-	OR
+ `^` 	-	Bitwise XoR
+ `~` 	-	Bitwise Not
+ `<<`	-	Left shift
+ `>>`	-	Right shift
